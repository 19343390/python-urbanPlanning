模板 Jinja2

参考《Flask Web 开发——基于Python的Web用于开发实战》、《深入理解Flask——掌握用Python创建强大动态Web应用技术》

模板是一个包含响应文本的文件，其中包含用占位变量表示的动态部分，其具体值只在请求的上下文中才能知道。使用真实值替换变量，再返回最终得到的响应字符串，这一过程称为渲染。为渲染模板，flask使用名为Jinjia2的模板引擎。
默认情况下，flask在程序文件夹中的templates子文件夹中寻找模板，因此HTML文件一般均放于此。HTML超文本标记语言(标准通用标记语言下的一个应用,Hyper Text Mark-up Language ),是迄今为止网络上应用最为广泛的语言,也是构成网页文档的主要语言。关于HTML语言的具体知识可以查询HTML 5.2 W3C Recommendation，以及自行网络搜索。

flask提供render_template('templateName',{key1:value1,key2:value2,...,keyn:valuen})函数把Jinjia2模板引擎集成到程序中。传入的键值对，key表示参数名，对应模板中的占位符，value为上下文中具体的值。

变量

Jinja2能识别所有类型的变量，例如列表、字典和对象，并在模板中使用{{name}}结构表示一个变量。例如：

<p>A value from a dictionary:{{dict['key']}}.</p>
<p>A value from a list:{{list[7]}}.</p>
<p>A value from a list,with a variable index:{{list[intvar]}}.</p>
<p>A value from an object's method:{{obj.somemethod()}}.</p>

可以使用过滤器修改变量，过滤器名添加在变量名之后，中间使用竖线分隔，例如项目中

var geoCoordMap = {{ imgLoc | safe}};
。Jinja2提供的部分常用过滤器：

过滤器名 说明
safe 渲染值时不转义。注：不要在不可信的值上使用safe过滤器，例如用户在表单中输入的文本
capitalize 把值的首字母转换成大写，其它字母转换成小写
lower 把值转换成小写形式
upper 把值转换成大写形式
title 把值中每个单词的首字母都转换成大写
trim 把值的首位空格去掉
striptags 渲染之前把值中所的HTML标签都删掉
default 如果传给它的值是None，则用一个默认值代替显示，如果希望变量仅为假值时替换默认值，传入参数True，例如：{{post.date | default('2018-02-02',True)}}
escape 如果传递的变量是HTML字符串，则其中的&、<、>、'和"被作为HTML转义序列打印
float 将传入值转换为浮点数显示，例如{{value | float}}
int 将传入值转换为整数显示
join 把列表拼成一个字符串，例如{{['flask','Jinja2'] | join(',')}}
length 同pythond的len()方法，计算列表或字符串长度
round 把浮点数转换到给定的小数位数，例如{{pi | round(2)}}
tojson 调研python的json.dumps函数序列化对象。
truncate 用于接收一个长字符串，返回一个截断成指定长度的字符串，并添加省略号

模板中的注释使用{# #}定义，不会出现在HTML中。

控制结构

控制语句的实际用法基本同python控制结构。过滤器可以用于控制语句中。
条件控制语句

{% if user %}
    Hello,{{ user }}
{% else %}
    Hello,Stranger!
{% endif %}

循环控制语句

<ul>
    {% for comment in comments %}
    <li>{{loop.index}}.{{ comment }}</li>
    {% endfor %}    
</ul>

在循环内部可以配合使用loop特殊变量获得关于循环的一些信息。具体如下：

变量 描述
loop.index 当前迭代项的序号(从1开始)
loop.index0 当前迭代项的序号(从0开始)
loop.revindex 当前迭代项的逆序序号(从1开始)
loop.revindex0 当前迭代项的逆序序号(从0开始)
loop.first 如果当前迭代项是迭代器中的第1个，则返回True
loop.last 如果当前迭代项是迭代器中的最后一个，则返回True
loop.length 迭代器的长度
loop.cycle 进行循环迭代的辅助函数，返回其参数中的下一个元素
loop.depth 表示递归循环中时的深度(从1开始)
loop.depth0 >表示递归循环中时的深度(从0开始)

宏控制语句
宏类似与python代码中的函数。

{% macro render_comment(comment) %}
    <li>{{ comment }}</li>
{% endmacro %}
<ul>
    {% for comment in comments %}
    <li>{{ render_comment(comment) }}</li>
    {% endfor %}
</ul>

如果重复使用宏，可以将其保存在单独的文件中，然后在需要使用的模板中导入：

{% import 'macros.html' as macros %}
<ul>
    {% for comment in comments %}
        {{ render_comment(comment) }}
    {% endfor %}
</ul>

include方法
需要在多处重复使用的模板代码片段可以写入单独的文件，在包含在所有模板中，避免重复：

{% include 'common.html' %}

模板继承
通过继承基础模板，替换基础模板中的标记段落，建立新模板。例如先建立基础模板base.html。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    {% block head %}
        <title>{% block title %}{% endblock %}-caDesign设计</title>
    {% endblock %}
</head>
<body>
    {% block body %}
    {% endblock %}
</body>
</html>

让后通过{% extends "base.html" %}方法基础模板，新模板为：

{% extends "base.html" %}
{% block title %}关于{% endblock %}
{% block head %}
    {{ super() }}
    <style>
    </style>
{% endblock %}
{% block body %}
    <h1>Hello,Flask!</h1>
{% endblock %}

链接

任何具有多个路由的程序都需要可以链接不同页面，flask提供了url_for()辅助函数，可以使用程序URL映射中保存的信息生成URL。

  1. 可以直接传入视图函数名，或者app.add_url_route()定义路由时使用的端点名作为参数，返回URL。例如：
        <p><a class="btn btn-primary btn-lg" href="{{ url_for('eval') }}" role="button">参与图像打分</a>
            &nbsp<a class="btn btn-primary btn-lg" href="{{ url_for('results') }}" role="button">查看评估结果</a>
            &nbsp<a class="btn btn-primary btn-lg" href="{{ url_for('imgprediction') }}" role="button">预测图像分数</a>
        </p>
  1. 如果调用 href="{{ url_for('eval'),_external=True}}返回的则是绝对地址。
  2. 使用url_for()生成动态地址时，将动态部分作为关键字参数传入。例如：url_for('user',name='caDesign',_external=True)，返回结果将是：http://localhost:5000/user/caDesign。
  3. 将任何额外参数添加到查询字符串中。例如：url_for('index',page=2)返回结果为：/?page=2。

使用flask-moment

服务器需要统一时间单位，与用户所在地理位置无关，一般使用协调世界时UTC(Coordinated Universal Time)。要在服务器上使用UTC时间，可以把时间单位发送给Web浏览器，转换成当地时间。flask-moment可以把moment.js集成到Jinjia2模板中。首先需要在terminal终端中安装pip install flask-moment，然后在主文件中调入并初始化。

from flask.ext.moment import Moment
import datetime
...
moment=Moment(app)

然后在HTML文档中引入，因为bootstrap已经引入了flask-moment依赖项jquery.js，因此在基模板中只需引入momnet.js。

{{ moment.include_moment() }}

在路由视图函数中，传入datetime变量。

@app.route('/')  
def index():
    return render_template('index.html',current_time=datetime.datetime.utcnow())

在对应的模板中使用flask-moment渲染时间戳。

<p>本地时间：{{ moment(current_time).format('LLL') }}</p>
<p>{{ moment(current_time).fromNow(refresh=True) }}</p>

format('LLL')根据客户端电脑中的时区和区域设置渲染日期和时间。fromNow(refresh=True)根据时间的推移自动刷新显示的时间。
